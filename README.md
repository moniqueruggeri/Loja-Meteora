# 🛍️ Loja Meteora

Projeto de um e-commerce responsivo desenvolvido com **React** e **Vite**, utilizando **Sass** para estilização. 

## 🚀 Tecnologias Utilizadas

- **React** + Vite ⚡
- **Sass** 🎨
- **JavaScript (ES6+)** 📝
- **Vite** 🛠️

# ✨ Funcionalidades
✅ Layout responsivo
✅ Design moderno e minimalista
✅ Organização modular com componentes React
